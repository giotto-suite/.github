# Giotto Suite 🔬✨

Welcome to the official repository home of **Giotto Suite**, a comprehensive and modular ecosystem of R packages engineered to effectively represent, analyze, and visualize spatial-omics data.

👉 **[www.giottosuite.com](https://www.giottosuite.com)**

## 📦 Ecosystem Overview

Giotto Suite is built out of distinct, highly optimized modules. This modular structure makes the ecosystem easy to extend and adapt for various high-throughput spatial technologies.

### Core Modules
* **[Giotto](https://github.com/giotto-suite/Giotto):** The flagship, umbrella spatial-omics toolbox built on top of the suite. It contains the primary functionalities for data processing, integration, spatial analyses, dimension reduction, and clustering.
* **[GiottoClass](https://github.com/giotto-suite/GiottoClass):** Houses the `giotto` object and subobject class definitions, ingestion workflows, data access, and core spatial frameworks. 
* **[GiottoVisuals](https://github.com/giotto-suite/GiottoVisuals):** The dedicated visualization engine, featuring dimension reduction, aggregated and subcellular spatial plots, color scale helpers, and plot output handlers.
* **[GiottoUtils](https://github.com/giotto-suite/GiottoUtils):** Handles internal infrastructure, package imports, and pretty printing. *(Mainly used internally; users rarely interact with this module directly).*

### Add-on Modules
* **[GiottoData](https://github.com/giotto-suite/GiottoData):** Provides mini example dataset objects for rapid testing, alongside utility functions to easily download and import publicly available datasets.
* **[GiottoDB](https://github.com/giotto-suite/GiottoDB) *(Experimental)*:** An active development framework designed to support scalable data analysis using on-disk database processing.
